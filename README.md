Player Survivor

Jogo 2D desenvolvido em Unity inspirado no estilo Vampire Survivors.

O jogador controla um olho que precisa sobreviver a hordas de inimigos enquanto dispara automaticamente contra outros olhos hostis.

Tecnologias utilizadas
Unity
C#
Programação Orientada a Objetos
Sistema de Spawn
IA básica de inimigos
Controle de colisão
HUD e Game Over
Funcionalidades
Movimento do jogador
Sistema de tiro automático
Progressão de dificuldade
Inimigos com perseguição
Sistema de pontuação
Gameplay survival arena
Objetivo do projeto

Projeto criado para estudos e prática de desenvolvimento de jogos utilizando Unity e C#.
